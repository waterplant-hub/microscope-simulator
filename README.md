# microscope-simulator
microscope simulator for students
# 複式顯微鏡操作模擬器

國一生物課使用的互動式顯微鏡操作練習工具。

## 使用方式

請開啟 GitHub Pages 網址：
https://waterplant-hub.github.io/microscope-simulator/

建議使用：
- iPhone / iPad：Safari
- Android：Chrome
- Chromebook / 電腦：Chrome

## 學習目標

1. 認識複式顯微鏡的基本構造
2. 練習對光、低倍對焦、移至視野中央、高倍細調
3. 理解目鏡視野上下左右相反
4. 理解玻片移動方向與視野影像移動方向相反
